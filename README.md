Project Title : Network Port Scanner - Port Sniffer

Tech Stack (Python Libraries):
1. Socket
2. Threading
3. Sys
4. Time
5. Tkinter
6. Fpdf2
7. G4f
8. Queue

This project is a user-friendly security tool designed to scan and identify open "doors" or ports on a specific computer or website using its IP address or hostname. 
While most scanners just list numbers, this application uses AI to explain what those open ports mean and whether they pose a security risk. 
To make the results useful for everyone, it automatically organizes these findings into a professional PDF report for formal records and a simple text file for quick 
technical reference. By putting these powerful features into a clear visual interface, it allows anyone to audit their network's safety and download a ready-to-use 
summary with just a few clicks.
